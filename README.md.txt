# 🫀 Heart Disease Prediction using Machine Learning

## 📌 Overview
This project applies Machine Learning techniques to predict heart disease based on patient medical data. Multiple classification algorithms are trained and evaluated to select the best performing model.

---

## ⚙️ Features
- Data cleaning and preprocessing
- Feature correlation analysis
- Multiple ML models comparison
- Confusion matrix visualization
- Prediction for new patient data

---

## 🤖 Models Used
- Logistic Regression  
- Support Vector Machine (Best Model)  
- Decision Tree  
- K-Nearest Neighbors  

---

## 🏆 Best Model
Support Vector Machine achieved the highest accuracy of **93%**

---

## 📊 Visualizations
- Correlation Heatmap  
- Confusion Matrix  

---

## 🛠️ Technologies
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  

---

## 📁 Files
- Python script (main project)
- Dataset (heart.csv)
- Visualization images

---

## 📌 Result
The system can predict whether a patient is likely to have heart disease based on input medical features.
